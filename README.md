# Poemly Prototype

This is a clickable frontend prototype for a social-media-like collaborative poetry platform.

Open `index.html` directly in a browser. The prototype is a self-contained offline file with embedded CSS and JavaScript, local mock data, and interactive state changes.

Suggested demo flow:

1. Browse **Home** to see a Poemly discover feed with starters, in-progress poems, and final versions.
2. Open a post and use familiar actions: like, comment, reply, save, quote, pin, and add to poem.
3. Use **Create** to publish a new poetic post with public/private and permission settings.
4. Use **Quote** to make your version while preserving the original source.
5. Use **Edit** to reorder or revise added lines, then publish or lock the version.
6. Use **History** to see how replies, quotes, and edits shaped the poem.
7. Explore **Collections**, **My Activity**, and **My Works** to see tags, participation, and profile history.
